Belajar Github dan Git CLI

1. git init === Inisialisasi Repositori
2. git clone <url_repositori> === Menbuat salinan Ke lokal
3. git add <nama_file1> <nama_file2> === menambahkan file ke staging area
4. git commit -m "Pesan komit" === melakukan komit
5. git status === melihat status
6. git log === commit hstory
7. git remote add origin <link-repo>
8. git push origin nama_branch
9. git pull origin nama_branch
10. git branch <nama_branch> === New Branch
11. git checkout <nama_branch> || git checkout <nama-branch> === beralih ke branch
12. git switch -c nama_branch === membuat branch dan pindah
13. git log --graph --oneline --all === log
14. --- === merge branch fitur to master
